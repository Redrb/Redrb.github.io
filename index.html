Mi enfoque abarca tanto el backend como el frontend, disfruto de programar, maquetar y refactorizar(en caso de ser necesario), muchas veces me encuentro tan inmerso que pierdo la noción del tiempo.

My knowledge

C#

Es el primer lenguaje aprendi y apliqué para entornos profesionales, hablar de este lenguaje me trae muy buenos recuerdos, cuando lo use fue para el backend de un proyecto de monitoreo de SSOMA, aqui aplique conocimientos de LINQ, inyeccion de dependecias, mis primeras nociones sobre migraciones, creacion de API seguras(utilizando jwt) ademas de un poco de frontend con razor

PHP

Debo admitir que al inicio no era de mis favoritos, sin embargo cuando se usa de manera ordenada, aplicando clean code y de la mano de algun framework como laravel se convierte en uno de mis preferidos, con php hice algunas integraciones con gmail y outlook.

Javascript

Cuando lo vi por primera vez me parecio un poco confuso ya que no es fuertemente tipado, en aquel momento incluso no se usaba las definiciones let y const, si no que todo se realizaba con definiciones del tipo var; Adicionalmente se agregaba undersore y jquery(actualmente ya son legacy) lo cual como novato en aque momento me confunció aun más, sin embargo en la actualidad y no es un problema, la animacion que ves al lado la hice usando js en pocos minutos

Dart

A mi parecer es un mezcla de c# con js, lo inicie a usar para poder realizar aplicaciones moviles y es bastante entretenido de aprender, me encanta la forma de usar las promesas muy similar a js, esto se acopla muy bien cuando se junta con Flutter.

Vue js

Fue el segundo framework js que vi y el que más usé; Usé options api y en el futuro pienso practicar composition api, de igual maner que con angulas la reactividad de los modelos me parece impresionante, nunca me voy a dejar de asombrar de esos aspectos, supongo que cuando vea react me pasará lo mismo. Desarrollé un editor html para personas que no tengan nociones de html, esto con el fin de maquetar documentos según las preferencias del usuario, no puedo mostrar capturas ya que fue para una empresa privada y considero que lo que haya desarrollado para mis anteriores trabajos es reservado.

Angular js

Es el primer framework js que usé, aqui obtube nociones de los modelos reactivos, ver como cambiaba algo en tiempo real en la interfaz de usuario me parecio algo increible y util, usar los loops facilitaba el trabajo y los watcher, para estar vigilando siempre si algún modelo cambiaba y agregar eventos en función al cambio que se esté realizando, mostrar alertas

Node js

Debido a que usa js me pareció bastante sencillo, mi principal uso de node js fue para realizar web scraping con puppeter, como recomendacion personal y debido a que se usan mayormente promesas, trata de hacer que tus funciones sean asincronas para usar async / await ya que si usas .then, para los casos de scrapping se van a anidar demasiado (por los tiempos de espera de elementos, navegaciones, etc.) y sería un poco problematico revisar el código, también puedese usar las anidaciones en el .then pero asegurate de sacar el contenido a una funcion para que mantengas el orden.
